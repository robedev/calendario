# Calendario Chile con Feriados

Aplicación web que genera calendarios mensuales con formato chileno, incluyendo todos los feriados nacionales y regionales de Chile.

## Características

- Calendario mensual con formato chileno (lunes a domingo)
- Feriados nacionales y regionales actualizados
- Ley de Puente implementada
- Generación de PDF para imprimir (formato carta, 2 páginas compactas con números de 46pt)
- Diseño responsivo y moderno
- Selección de región para feriados regionales

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Tecnologías

- React 18
- Vite
- Tailwind CSS
- date-fns
- jsPDF
